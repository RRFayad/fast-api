#### Lists

- Same as arrays in JS, some obs and methods:
  - `list_name[-1]` gets last item (or [-X] will get last x item)
  - `list_name[0:2]` gets the slice from 0 up to 2 NOT included (and it does not mutate the list)
  - `list_name.append()`
  - `list_name.remove()`
  - `list_name.pop()`
  - `list_name.reverse()`
  - `list_name.sort()`
